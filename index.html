Workflow

How the prototype classifies an item

  1. 1 Image upload

    The user provides a photo of one waste item.

  2. 2 Preprocessing

    The image is resized and normalized for model input.

  3. 3 Feature extraction

    Visual patterns such as color, brightness, and texture are analyzed.

  4. 4 Classification

    The app returns recyclable or non-recyclable with confidence.

Project Context

Decision-support tool for recycling

This application demonstrates the interface and inference workflow for a CNN-based waste classifier trained on categories similar to the TrashNet dataset. A production version can replace the local demo logic with exported CNN weights from TensorFlow, Keras, or TensorFlow.js.

Recyclable

Paper, cardboard, glass, metal, and most plastic containers.

Non-recyclable

Food waste, contaminated items, mixed trash, and low-confidence unknown objects.
